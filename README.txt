SO utilizado: Linux (Ubuntu 16.04)

Modo de compilação: g++ -ansi -Wall -std=c++11 Tradutor.cpp -o Tradutor (não necessariamente Tradutor, pode ser qualquer nome)

Modo de execução: No terminal do linux digitar ./Tradutor [arquivo].asm (Não necessariamente Tradutor, mas o nome escolhido na compilação)
*[arquivo].asm é o arquivo de teste em assembly inventado (ex: Teste.asm)
